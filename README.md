# Retail Sales Analytics: SQL + Tableau Dashboard

## 📌 Overview
This project demonstrates an end-to-end Business Intelligence workflow:
- MySQL database design and SQL queries
- CSV exports for data portability
- Tableau dashboards for interactive analytics

## 🗂️ Dataset
- **Customers**: Customer demographics and regions
- **Orders**: Order details with product references
- **Products**: Product categories and pricing
- **Payments**: Transaction records linked to orders

## 📊 Dashboard Features
- Revenue trend analysis over time
- Top customers by sales contribution
- Product category performance (pie chart)
- Regional insights with interactive filters

## 🛠️ Tech Stack
- MySQL (database + queries)
- SQL (schema, joins, aggregations)
- Tableau (visualizations + dashboards)

## 🚀 How to Run
1. Clone this repository
2. Load CSV files into Tableau
3. Open `tableau_dashboard.twbx`
4. Explore interactive visuals
